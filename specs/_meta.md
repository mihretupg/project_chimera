# Project Chimera Meta Specifications

## Non-goals
- The system will **not generate content autonomously** without agent orchestration.
- No direct API access to social media endpoints.
- Will not store personal user data outside metadata.

## Constraints
- Budget constraints: low-cost hosting and storage.
- Safety constraints: all outputs must be verified before external publication.
- No direct human editing except HITL escalation.

## Out of scope for Day 3
- Vector DB implementation
- Full CI/CD pipelines
- Video embedding search
